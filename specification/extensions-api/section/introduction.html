Introduction

This specification defines an extensions API that allows browser extensions to manage a browser's headless [[WebMonetization]] implementation, as described in [[WebMonetizationFlows]]. The browser implements all core monetization functionality — processing monetization links, managing sessions, handling payment flows, and communicating with wallets — while extensions provide the management UI and apply user-tailored customization on top of the browser-provided engine.

This specification is a companion to [[WebMonetization]] and [[WebMonetizationFlows]]. It does not expose payment credentials or private wallet data.
