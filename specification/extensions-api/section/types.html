Types

MonetizationInfo

Describes the document where a monetization state change occurred. 

      dictionary MonetizationInfo {
        required long tabId;
        required long frameId;
        required DOMString url;
      };

Properties

tabId
The identifier of the tab containing the monetized document.
frameId
The identifier of the frame within the tab. 0 denotes the top-level (main) frame.
url
The URL of the monetized document.
