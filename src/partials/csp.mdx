If your website uses Content Security Policy headers, you must add the following directives to allow the publisher tools embed script to work:

| Directive | Value | Description |
| --------- | --- | ----------- |
| <span style="white-space:nowrap">`script-src`</span> | <span style="white-space:nowrap">`https://tools-cdn.webmonetization.org`</span> | Allows loading interactive tool script from the CDN |
| <span style="white-space:nowrap">`connect-src`</span> | <span style="white-space:nowrap">`https://tools-api.webmonetization.org`</span> | Allows fetching configuration data from the API |
| <span style="white-space:nowrap">`img-src`</span> | <span style="white-space:nowrap">`data: https://tools-cdn.webmonetization.org`</span> | Allows loading logos/thumbnails and inline SVG icons |
| <span style="white-space:nowrap">`style-src`</span> | <span style="white-space:nowrap">`https://tools-cdn.webmonetization.org`</span> | Allows loading CSS files and dynamic theme styling (custom colors/fonts) |
| <span style="white-space:nowrap">`font-src`</span> | <span style="white-space:nowrap">`https://tools-cdn.webmonetization.org`</span> | Allows loading custom web fonts |
