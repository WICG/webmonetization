## Step 1 - Get your payment URL 

### Wallet address

If your wallet provided you with a wallet address in the format below, proceed to [Step 2](#step-2---web-monetize-your-page). 

```http
https://wallet.example.com/alice
```

### Payment pointer

If your wallet provided you with a payment pointer in the format below, visit <LinkOut href='https://paymentpointers.org/'>paymentpointers.org</LinkOut>.

```http
$wallet.example.com/alice
```
Enter your payment pointer into the _Payment Pointer_ field. Take note of the URL that automatically generates in the adjacent field. You'll need it in Step 2. 

## Step 2 - Web monetize your page

<Steps>

1. Create your Web Monetization link tag using the format below. Add your payment URL as the `href` value.

    ```http
    <link rel="monetization" href="your_url">
    ```

    For example:

    ```http
    <link rel="monetization" href="https://wallet.example.com/alice">
    ```

2. Add the `<link>` to the `<head>` section of your page.
    ```html title="Example"
    <html>
      <head>
        <title>My Site</title>
        <link rel="monetization" href="https://wallet.example.com/alice">
      </head>
    </html>
    ```
</Steps>

Congratulations! Your page is now web monetized. Add the `<link>` to any page of your site you want to monetize. 

For more information, visit the [Monetization link element](/experiment/link-element) page.
