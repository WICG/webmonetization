The `monetization` event's `amountSent` attribute returns you the amount (`value`) and <Tooltip content='The ISO 4217 alphabetic code that represents a currency (USD, CAD, etc.)' client:load><span>currency code</span></Tooltip> (`currency`) of the Web Monetization payment sent to you.

## Value

The `value` uses the same data structure as the <LinkOut href="https://www.w3.org/TR/payment-request/#dom-paymentcurrencyamount">`PaymentCurrencyAmount`</LinkOut> dictionary in the <LinkOut href="https://www.w3.org/TR/payment-request/">Payment Request API</LinkOut>. The returned value is the value it was initialized with.

## Example

```html
<link rel="monetization" href="https://wallet.example/alice" />
<script>
  const link = document.querySelector('link[rel="monetization"]')
  link.addEventListener('monetization', (event) => {
    // See how much was sent and in what currency
    const {
      amountSent: { value, currency },
    } = event
    console.log(`Browser sent ${currency} ${value}.`)
  })
</script>
```
