The `monetization` directive within the HTTP <LinkOut href="https://developer.mozilla.org/en-US/docs/Web/HTTP/Headers/Permissions-Policy">Permissions-Policy</LinkOut> header controls whether the current document and any nesting browsing contexts (iframes) are allowed to use the Web Monetization API.

## Syntax

The basic syntax is as follows, where `allowlist` is a list of origins permitted to use Web Monetization.

```http
Permissions-Policy: monetization=(allowlist)
```

## Example

Allow Web Monetization in the document and in all nested browsing contexts (iframes) in the same origin.

```http
Permissions-Policy: monetization=(self)
```

This is the default policy for `monetization`.

:::note
The `allowlist` attributes only take effect when the _content navigable_ of the iframe is navigated. Adding or removing the monetization attribute has no effect on an already-loaded document.
:::
