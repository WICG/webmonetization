Google's Offerwall is a monetization tool that gives publishers more ways to earn beyond traditional advertisements. For example, a visitor could choose between watching a short ad or providing their email address before accessing content.

This page describes how to use the <LinkOut href="https://webmonetization.org/tools/offerwall">Offerwall tool</LinkOut> and Google Ad Manager to add Web Monetization to your Offerwall.

### Before you begin

- You must have a Google Ad Manager account.
- You can only add one custom user choice to a Google Offerwall.

### Generate your script

1. Click **Save and generate script**.<br /><img src="/img/docs/pubtools/tools-save-buttons.png" alt="The save edits only and the save and generate script buttons side by side" style="max-width: 45%" class='img-outline' />
   :::note[Save edits only]
   Click **Save edits only** if you aren't ready to generate your script. Your configurations are saved on webmonetization.org.
   :::
2. Add the script to the `<body>` of each page you want to offer Web Monetization.
   ```js wrap title="Example script"
   <script
     id="wmt-offerwall-init-script"
     type="module"
     src="https://tools-cdn.webmonetization.org/offerwall.js"
     data-wallet-address="https://sandbox.ilp.link/rabeb"
     data-tag="version1"
   ></script>
   ```

## Add Web Monetization as a custom Offerwall choice

1. In Google Ad Manager, select the _Privacy & Messaging_ tab.
2. Select _Offerwall_ from the list of Monetization options.
3. <LinkOut href="https://support.google.com/admanager/answer/11897778">
     Create an Offerwall message
   </LinkOut>
   .
4. Turn on _Custom choice_ in the _User choices_ section.
5. <LinkOut href="https://support.google.com/admanager/answer/13566866">
     Set up a "custom choice" user choice
   </LinkOut>
   .
6. **We should put recommendations here for a custom choice title and subtitle. The screenshot should reflect the recommendations. For example, "Web Monetization" and "Support us with micropayments".**
   <img
     src="/img/docs/pubtools/offerwall-choice-list.png"
     alt="Example user choice screen that includes Web Monetization as an option"
     style="max-width: 60%"
     class="img-outline"
   />
7. Publish your Offerwall.
8. Go to your site and check that the Offerwall appears correctly. If your Offerwall doesn't appear at all, check that you've enabled the custom choice within Ad Manager with the correct implementation. To fully test the Web Monetization choice, you'll need to visit your site with the extension:
   - Uninstalled
   - Installed but disabled
   - Installed and enabled

## Change an existing appearance configuration

1. Go to the <LinkOut href="https://webmonetization.org/tools/offerwall">Offerwall tool</LinkOut> and enter your wallet address.
2. Click **Continue**. If a confirmation prompt appears, you must first confirm your ownership of the wallet address.
3. Make your changes, then click **Save edits only**. Your [script](#generate-your-script) remains the same. There's no need to update your site's code.
4. Refresh a page on your site that uses the configuration's script to verify your changes are live.

## Connect to a new wallet address

Use the <LinkOut href="https://webmonetization.org/tools/offerwall">Offerwall tool</LinkOut> to change the wallet address that receives your Web Monetization contributions.

<Change />

All pages that include the script are now web monetized with the new wallet address.
